# Personal Website

## Development

`yarn install`

### Front End

`yarn dev` - Runs frontend on localhost.

`yarn build` - Builds frontend with vite, output into `/dist` folder

`yarn preview` - Runs local static web server using build from `/dist` folder.

### Back End

`yarn server-dev` - Runs dev mode of express webserver on localhost.

## Deployment

1. Clone repository
2. `yarn install`
3. `yarn build`
4. `yarn deploy`
